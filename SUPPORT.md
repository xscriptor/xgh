# Support

If you need help with this repository, here are a few ways to get it:

- **Issues**: If you've found a bug or have a suggestion for a new configuration, please open an issue in this repository.
- **Pull Requests**: Have a fix or a new workflow to share? We welcome Pull Requests! Make sure to read the `CONTRIBUTING.md` first.
- **Email**: For specific or private inquiries, please contact x@xscriptor.com.
